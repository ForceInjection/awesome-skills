# Supported Ratios

Use these presets for editorial cards and covers.

| Ratio    | Use case                | Viewport size |
| -------- | ----------------------- | ------------- |
| `3:4`    | Portrait info card      | `1500 x 2000` |
| `4:3`    | Landscape info card     | `2000 x 1500` |
| `1:1`    | Square post             | `1800 x 1800` |
| `16:9`   | Standard wide cover     | `1920 x 1080` |
| `9:16`   | Story / reel cover      | `1080 x 1920` |
| `2.35:1` | Cinematic cover         | `2350 x 1000` |
| `3:1`    | Profile / banner cover  | `1800 x 600`  |
| `5:2`    | Ultra-wide strip cover  | `2500 x 1000` |

## General

- Use the preset pixel dimensions as the actual design canvas, not only as the browser window size.
- Keep the page itself aligned to the target ratio so the screenshot does not need cropping.
- Rebalance the composition for each ratio. Do not treat ratios as simple viewport resizes of one fixed template.
- `16:9` and `9:16` are inverse orientations of the same aspect relationship; content density strategy differs, not just rotation.

## Per Ratio

- **`4:3`** — asymmetric left-right; heavy primary + narrower judgment stack + thin footer.
- **`3:4`** — cover-style hero + mixed-scale modules; do not default to a single narrow column.
- **`1:1`** — one heavy quadrant + supporting blocks; avoid four equal tiles.
- **`16:9`** / **`9:16`** — fewer, larger blocks; aggressive paragraph reduction.
- **`2.35:1`** / **`5:2`** — ultra-wide; reduce paragraph length, use fewer but larger blocks.
- **`3:1`** — keep text sparse, protect likely avatar overlap zones, avoid filling the full width with copy.
